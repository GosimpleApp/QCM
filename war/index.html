L'outil de QCM Open Source

Bonjour

Voici un outil libre pour la génération et l'utilisation de QCM, en Version Beta.

Pour :

+ saisir des QCM sur vos cours,

+ les transmettre à vos élèves pour les jouer

Vos QCM sont publiés à votre demande, ou bien accessibles par tout le monde.

L'outil de génération

Il vous permet d'organiser et de sasir vos QCM, avec un classement par enseignement, matières et cours

Accès (mode test) »

Le player

Chargez un QCM à l'aide son code, puis jouez le en ligne, ou hors ligne, sur un ordinateur ou un smartphone.

Le code

Cette suite est open source, écrit en java avec GWT, il utilise un accès configurable à une base de données de votre choix.
